Overview

The Applied Data Science program is designed to develop students across technical, analytical, and ethical dimensions of data science. The program learning outcomes emphasize six key areas: collecting and managing data using appropriate technologies, generating actionable insights across different contexts, applying visualization and predictive models, using programming languages such as Python and R, communicating insights effectively to diverse audiences, and applying ethical considerations such as fairness, bias, and privacy in data-driven solutions.

My portfolio includes projects that collectively demonstrate these outcomes across multiple domains of data science. One of the key projects is a transformer fine tuning and semantic drift analysis project using CEO earnings call transcripts. This project involved building an end to end NLP pipeline that extracted forward looking statements, applied weak labeling using large language models, and fine tuned a transformer based encoder to improve representation learning. The project also tracked how themes evolved over time, providing insight into changes in corporate communication.

Another project focused on real time sentiment analysis of U.S. election data using Twitter. This project included data collection, preprocessing, feature engineering, and model development using both traditional machine learning and deep learning techniques. It also involved visualizing sentiment trends over time, demonstrating how data can capture shifts in public opinion in response to real world events.

In addition, I worked on a computer vision project that used a YOLOv8 model to detect and classify retail products. This project involved dataset creation, annotation, model training, and evaluation, and highlighted challenges such as class imbalance and generalization. I also completed a credit card fraud detection project using Support Vector Machines in R, where I addressed class imbalance and evaluated model performance using metrics such as accuracy, recall, and AUC.

These projects demonstrate how I achieved the program learning outcomes. Data collection and processing were central to all projects, addressing the first outcome. The full data science lifecycle was applied in each case, supporting the second outcome. Predictive modeling and visualization techniques were used extensively, fulfilling the third outcome. Programming in Python and R was essential across projects, meeting the fourth outcome. Visualization and reporting enabled effective communication of insights, addressing the fifth outcome. Finally, understanding model limitations and data bias helped incorporate ethical considerations, fulfilling the sixth outcome.

Overall, my portfolio reflects a comprehensive application of data science concepts, combining technical skills with practical problem solving and responsible use of data.
